Modelo de Teachable Machine

La inteligencia artificial (IA) es la capacidad de las máquinas para imitar la inteligencia humana, como aprender, razonar y tomar decisiones. ¿Qué es un modelo de IA? Un modelo de IA es un algoritmo que ha sido entrenado con datos para realizar tareas específicas, como clasificar imágenes o predecir resultados. Teachable Machine: Es una herramienta de Google que permite crear modelos de IA sin necesidad de programar. En este projecto aprendemos a entrenar modelos para reconocer imágenes, sonidos o poses utilizando los datos que elijamos. Se entrenó un modelo de reconocimiento de imágenes con Teachable Machine para identificar clases visuales mediante la cámara. El modelo usa IA para clasificar en tiempo real lo que ve el usuario. Intenta identificar carecteristicas cercanas a las que se entrenó y haci poder darles un porcentaje de lo que cree que es. Mis clases son 3: Mano, moneda, celular. El modelo se entrena con imágenes de cada clase y luego se prueba con nuevas imágenes para evaluar su precisión.
